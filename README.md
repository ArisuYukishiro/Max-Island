# MaxIsland

Tired of switching tabs to understand a sentence or word during immersion? MaxIsland sits at the top of your screen — ask AI instantly without breaking your focus.

## What it does

A small floating island lives at the top center of your screen. Click to expand, ask about any word or sentence, get an answer, then collapse and keep going. No tab switching, no context loss.

## Setup

Copy the config and fill in your API details:

```bash
cp MaxIsland/Config/Secrets.xcconfig.example MaxIsland/Config/Secrets.xcconfig
```

Then edit `Secrets.xcconfig` with your API base URL.

## Requirements

- macOS 26.0+
- Xcode 26.1.1+
